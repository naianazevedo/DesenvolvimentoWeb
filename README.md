# Desenvolvimento Web - Uninter

Neste repositório estão os materiais das aulas da disciplina de Desenvolvimento Web (Uninter).

O que será aprendido nas aulas:

- Aula 1: Estrutura básica de HTML (tags, cabeçalho, parágrafos)
- Aula 2: Listas, links e imagens
- Aula 3: Tabelas e formulários
- Aula 4: Introdução a CSS (seletores, cores, fontes)
- Aula 5: Layouts com Flexbox e Grid
- Aula 6: Responsividade e media queries
- Aula 7: Introdução a JavaScript e manipulação do DOM
- Aula 8: Projeto prático e publicação (ex.: GitHub Pages)

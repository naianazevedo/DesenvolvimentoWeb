#Aula 2 - CSS
Formatações básicas em CSS, que servem de base para o desenvolvimento de aplicações web.
Foram abordados sobre o Cascading Style Sheets (CSS): definição, sintaxe, seletores e como aplicar CSS e hierarquia CSS.
Além de comentários em CSS, cores, fundo (background), bordas, margin e preenchimento (padding).
Abordou-se também propriedades de largura (width), altura (height), formatação de texto/fonte e modelo de caixa (box model). 
E também float, clear, elementos block, inline, display, position, z-index e overflow.

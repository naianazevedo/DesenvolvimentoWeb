# Desenvolvimento Web
Neste repositório estão os materiais das aulas da disciplina de Desenvolvimento Web.

O que será aprendido nas aulas:

Aula 1: Estrutura básica de HTML (tags, cabeçalho, parágrafos e etc.)
Aula 2: Formatação básica em CSS (sintaxe, seletores, background, padding)
Aula 3: Aprofundamento em CSS (DOM, peseudoclasses e pseudoelementos)
Aula 4: HTML Semântico e CSS (Elementos estruturais semânticos, grid, flexbox)
Aula 5: Introdução ao JAVASCRIPT (História, hospedagens, variáveis e operadores)
Aula 6: Aprofundamento em JAVASCRIPT (Objetos, arrays, funções, BOM e DOM)
Aula 7: Atividade prática (Desenvolvimento de um portfólio online pessoal)

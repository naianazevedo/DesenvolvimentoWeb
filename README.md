# Uninter - Disciplina de Desenvolvimento Web

Bem-vindo(a) à disciplina de Desenvolvimento Web da Uninter! Neste repositório você encontrará os materiais de apoio, exercícios e exemplos de código usados nas aulas para aprender a construir sites e aplicações web front-end.

## Ementa da Disciplina
A disciplina aborda conceitos fundamentais e práticos do desenvolvimento web, com foco em:

- Estrutura e semântica de documentos com HTML
- Estilização e layout usando CSS (incluindo responsividade)
- Interatividade com JavaScript e manipulação do DOM
- Boas práticas de acessibilidade e SEO básico
- Uso de ferramentas de desenvolvimento (Visual Studio Code, inspeção do navegador)
- Versionamento de código com Git e GitHub

## Conteúdo Programático
1. Introdução ao desenvolvimento web e ambiente de trabalho
2. HTML: tags básicas, estrutura, elementos semânticos, listas, tabelas e formulários
3. CSS: seletores, propriedades, box model, posicionamento, flexbox, grid e media queries
4. JavaScript: sintaxe básica, variáveis, funções, eventos e DOM
5. Trabalhando com formulários e validação
6. Criação de layouts responsivos e mobile-first
7. Publicação e hospedagem simples (GitHub Pages)

## Objetivos
Ao final da disciplina, o(a) aluno(a) deverá ser capaz de:

- Construir páginas estáticas bem-estruturadas em HTML
- Aplicar estilos e criar layouts responsivos com CSS
- Adicionar interatividade básica usando JavaScript
- Utilizar ferramentas de desenvolvimento e versionamento de código

## Ferramentas e Tecnologias
- Editor: Visual Studio Code
- Linguagens: HTML, CSS e JavaScript
- Controle de versão: Git e GitHub

## Organização do Repositório
Cada aula possui uma branch própria (por exemplo: `Aula-1`, `Aula-2`, `Aula-3`) contendo os arquivos e exemplos correspondentes. Use essas branches para acessar o conteúdo de cada encontro.

## Como Contribuir
Sinta-se à vontade para enviar issues com dúvidas ou sugestões. Para contribuir com código, crie uma branch a partir da branch da aula correspondente, faça as alterações e abra um Pull Request.
